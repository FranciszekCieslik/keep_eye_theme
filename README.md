# Keep Eye
Keep Eye is visual studio code dark theme.

## Installation
Launch VS Code Quick Open (Ctrl+P), paste the following command, and press enter.

```Microsoft Visual Studio Solution
ext install HupanCoder.keep-eye-theme
```
